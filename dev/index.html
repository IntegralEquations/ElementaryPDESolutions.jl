PolynomialSolutions

Generate polynomial solutions to certain PDEs given a polynomial source term.

Helmholtz equation

The function solve_helmholtz takes a Polynomial Q and returns the unique Polynomial P satisfying

\[ \Delta P + P = Q\]

Some solutions in 2D:

using PolynomialSolutions
for j in 0:4, i in 0:4
    Q = monomial(i,j)
    P = solve_helmholtz(Q)
    @show P, Q
end
(P, Q) = (1, 1)
(P, Q) = (x, x)
(P, Q) = (-2 + x², x²)
(P, Q) = (-6x + x³, x³)
(P, Q) = (24 - -12x² + x⁴, x⁴)
(P, Q) = (y, y)
(P, Q) = (xy, xy)
(P, Q) = (-2y + x²y, x²y)
(P, Q) = (-6xy + x³y, x³y)
(P, Q) = (24y - -12x²y + x⁴y, x⁴y)
(P, Q) = (-2 + y², y²)
(P, Q) = (-2x + xy², xy²)
(P, Q) = (-2y² - -2x² + x²y², x²y²)
(P, Q) = (-6xy² - -2x³ + x³y², x³y²)
(P, Q) = (24y² + 48x² - -12x²y² - -2x⁴ + x⁴y², x⁴y²)
(P, Q) = (-6y + y³, y³)
(P, Q) = (-6xy + xy³, xy³)
(P, Q) = (-2y³ - -6x²y + x²y³, x²y³)
(P, Q) = (-6xy³ - -6x³y + x³y³, x³y³)
(P, Q) = (24y³ + 144x²y - -12x²y³ - -6x⁴y + x⁴y³, x⁴y³)
(P, Q) = (24 - -12y² + y⁴, y⁴)
