PolynomialSolutions

Generate polynomial solutions to certain PDEs given a polynomial source term.

Helmholtz

The function solve_helmholtz takes a Polynomial Q and returns the unique Polynomial P satisfying

\[ \Delta P + P = Q\]

Laplace

The function solve_laplace takes a Polynomial Q and returns a Polynomial P satisfying

\[ \Delta P = Q\]

Some solutions in 2D:

using PolynomialSolutions
for j in 0:4, i in 0:4
    Q = monomial(i,j)
    P = solve_laplace(Q)
    @show P, Q
end

The function solve_stokes takes an NTuple of Polynomials Q and returns two Polynomials, U and P, satisfying

\[ \begin{align*} \Delta U - \nabla P &= Q \\ \nabla \cdot U &= 0 \end{align*}\]
